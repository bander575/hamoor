# هامور البحر

لعبة صيد أسماك عربية تعمل في المتصفح فقط، مبنية بـ HTML / CSS / JavaScript / Phaser.

## التشغيل المحلي

افتح `index.html` من سيرفر محلي:

```powershell
python -m http.server 8765
```

ثم افتح:

```text
http://127.0.0.1:8765/
```

## اللعب

- تحرك بالأسهم أو WASD.
- اقترب من القارب واضغط `E` أو زر "ركوب القارب".
- اذهب لمنطقة صيد واضغط `F` أو زر "رمي الشبكة".
- انزل عند التاجر واضغط زر "بيع الكل".
