# هامور البحر - شاشة الميناء

شاشة ميناء عربية تعمل في المتصفح فقط، مبنية بـ HTML / CSS / JavaScript / Phaser.

## التشغيل المحلي

افتح `index.html` من سيرفر محلي:

```powershell
python -m http.server 8765
```

ثم افتح:

```text
http://127.0.0.1:8765/
```

## المحتوى الحالي

- تحرك بالأسهم أو WASD.
- تحدث مع الصيادين بالضغط على `E`.
- ادخل سوق السمك أو محل القوارب أو المستودع أو المنارة.
- لا يوجد صيد أو جزر أو ترقيات في هذه المرحلة.
